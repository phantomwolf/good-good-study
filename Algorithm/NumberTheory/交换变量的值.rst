交换两个变量的值
=========================
临时变量法
---------------
最普通也是最快的方法::

	register int tmp = x;
	x = y;
	y = tmp;


异或法XOR
---------------
使用位运算XOR::

	int x = 0x0001, y = 0x0011;
	x ^= y;		// x: 0x0010, y: 0x0011
	y ^= x; 	// x: 0x0010, y: 0x0001
	x ^= y; 	// x: 0x0011, y: 0x0001

虽然不需要临时变量，但实际上性能不如临时变量法。


数学运算法
---------------
使用数学运算也可以，比如减法::

	x = x + y;	// x: x+y, y: y
	y = x - y;	// x: x+y, y: x
	x = x - y;	// x: y, y: x

甚至乘除法::

	x = x * y;	// x: x*y, y: y
	y = x / y;	// x: x*y, y: x
	x = x / y;	// x: y, y: x

但是这样性能较差(算术运算显然不如位运算快)，而且还有溢出的风险。对于乘除法，如果其中一个数为0，还会导致异常。
